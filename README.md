Proyecto movil
================

Este repositorio contiene una aplicacion movil construida con Expo y React Native.

Requisitos previos
-----------------
- Node.js 18 o superior
- npm 9 o superior (o pnpm/yarn si prefieres)
- Aplicacion Expo Go instalada en tu dispositivo movil para las pruebas rapidas

Instalacion
-----------
```bash
npm install
```

Ejecucion en modo desarrollo
----------------------------
```bash
npm run start
```

Esto abrira Expo CLI; puedes escanear el codigo QR con Expo Go o presionar `a`, `i` o `w` para abrir el proyecto en Android, iOS o web respectivamente.

Estructura del proyecto
-----------------------
- `components/`: Componentes reutilizables de la interfaz.
- `screens/`: Pantallas principales de la aplicacion.
- `assets/`: Recursos estaticos como imagenes e iconos.
- `constants/`: Configuracion compartida como temas y estilos.

Personalizacion
---------------
- Actualiza `app.json` para modificar el nombre, icono o informacion del proyecto.
- Revisa los componentes en `components/ui/` para ajustar la identidad visual.

Scripts disponibles
-------------------
- `npm run start`: Inicia el servidor de desarrollo de Expo.
- `npm run android`: Lanza la aplicacion en un emulador o dispositivo Android.
- `npm run ios`: Lanza la aplicacion en un simulador de iOS (macOS requerido).
- `npm run web`: Ejecuta el proyecto en el navegador.

Soporte y recursos
------------------
- Documentacion de Expo: https://docs.expo.dev/
- Documentacion de React Native: https://reactnative.dev/docs/getting-started
