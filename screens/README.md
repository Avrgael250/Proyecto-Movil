# Paleta de Colores - Aplicación Financiera

## Colores Base
- **Fondo Principal**: `#E3F2FD` (Azul muy claro)
- **Fondo Secundario**: `#FFFFFF` (Blanco)
- **Color Principal**: `#4A8FE7` (Azul principal)
- **Color Hover**: `#3A7FD7` (Estado interactivo)
- **Texto Principal**: `#030213` (Negro suave)
- **Texto Secundario**: `#666666` (Gris)
- **Texto Terciario**: `#717182` (Gris medio)

## Elementos de Navegación
- **Barra de Navegación**: 
  - Fondo: `#E3F2FD`
  - Borde Superior: `#d1d1d1`
  - Íconos Activos: `#4A8FE7`
  - Íconos Inactivos: `#030213`

## Botones y Acciones
- **Botón Flotante (+)**:
  - Fondo: `#4A8FE7`
  - Texto/Ícono: `#FFFFFF`
  - Sombra: `rgba(0, 0, 0, 0.3)`
  
- **Botón de Transferencias**:
  - Fondo: `#0EA5E9`
  - Texto: `#FFFFFF`

- **Botón "Este mes"**:
  - Fondo: `#E3F2FD`
  - Texto: `#4A8FE7`

## Tarjetas y Contenedores
- **Tarjetas**:
  - Fondo: `#FFFFFF`
  - Borde/Separador: `#F0F0F0`
  - Sombra: `rgba(0, 0, 0, 0.1)`

## Categorías de Gastos

  - Ícono Fondo: `#2196F3`
  - Barra de Progreso: `#2196F3`

- **Transporte**:
  - Ícono Fondo: `#FF9800`
  - Barra de Progreso: `#FF9800`

- **Servicios**:
  - Ícono Fondo: `#9C27B0`

## Gráficos y Visualización de Datos

  - Amarillo: `#F4D03F`

- **Gráfico de Barras**:
  - Azul: `#5DADE2`

## Estados y Feedback
- **Tab Activo**: 
  - Borde Inferior: `#2196F3`
  - Texto: `#2196F3`

- **Tab Inactivo**:
  - Texto: `#666666`

## Elementos Interactivos
- **Enlaces y Acciones**:
  - Color Principal: `#4A8FE7`
  - Color Secundario: `#2196F3`

## Sistema de Sombras
- **Sombra Estándar**:
  ```css
  shadowColor: '#000000'
  shadowOffset: { width: 0, height: 2 }
  shadowOpacity: 0.1
  shadowRadius: 4
  elevation: 2
  ```

- **Sombra Pronunciada** (Botones flotantes):
  ```css
  shadowColor: '#000000'
  shadowOffset: { width: 0, height: 4 }
  shadowOpacity: 0.3
  shadowRadius: 8
  elevation: 8
  ```
